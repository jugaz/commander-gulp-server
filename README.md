### Instalación

```bash
$ npm install commander-gulp-server
```

### Comando a ejecutar

```bash
$ commander-gulp-server server 'entry' --ser 'ouput' 
```
### Configuración por el package.json

```bash
"scripts": {
  "server": "commander-gulp-server server \"8080\" --ser \"docs/\""
}
```
