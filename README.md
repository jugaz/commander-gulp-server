## Installation

```bash
$ npm install commander-gulp-server
```


#### Command to Compile

```bash
$ commander-server server 'entry' --ser 'ouput' 
```
#### Example

```bash
 "scripts": {
    "server": "commander-gulp-server server \"8000\" --ser \"docs/\"",
  }
```
